# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Contact:** slava AT claritypledge DOT com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

We will acknowledge receipt within 48 hours and work with you to understand and address the issue.

## Scope

This policy applies to the Understanding Pledge web application and its associated codebase.
