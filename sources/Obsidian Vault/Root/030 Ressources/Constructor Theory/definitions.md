• **Good Explanation**

Hard-to-vary, falsifiable, counterfactual, antifragile, recursively teachable, self-justifying, scalable.

• **Biggest Double Crux**

Disagreement on criteria defining a **Good Explanation**, solving which enables maximum future agreement.

• **Epistemic Turing Test (ETT)**

Verifies an agent’s ability to **recursively** teach others how to solve the **Biggest Double Crux**.

• **Universal Epistemic Constructor (UEC)**

Agent who helps others pass the **ETT**, thus **universally** resolving the **Biggest Double Crux**.
